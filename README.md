# Keycloak Microfrontends (Module Federation)

## Apps
- shell (http://localhost:3000)
- users-list (http://localhost:3001)
- users-form (http://localhost:3002)

## Run each app
cd shell && npm start
cd users-list && npm start
cd users-form && npm start

## Keycloak
Realm: master
Client: frontend-app
Grant type: password
